add button to downlod the app : # 🕌 Adhan Prayer Times App (Android)

A modern Android application that provides accurate prayer times, Adhan notifications, and offline support for Morocco 🇲🇦 .

---

## ✨ Features

### ⏰ Prayer Times

* Daily prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha)
* Next prayer detection
* Real-time countdown timer

### 🔊 Adhan Notifications

* Automatic Adhan at prayer time
* Mute / Volume control directly from notification
* Works even when the app is closed

### 🌐 City Selection

* Choose Moroccan cities (Ifrane, Fes, Rabat, Casablanca…)
* Smooth bottom-sheet UI (premium style)

### 🌙 Dark Mode

* Fully supported (auto + manual)

### 🎨 Premium UI

* Glassmorphism prayer card
* Background image support
* Arabic-friendly design

---

## 📴 Offline Mode (Planned / In Progress)

* Store full month prayer data using Room (SQLite)
* Access prayer times without internet
* Smart sync when online

---

## 🌍 Languages

* Arabic 🇸🇦 (RTL support)

## 🧠 Architecture

* MVVM (Model - View - ViewModel)
* Repository pattern
* LiveData for reactive UI
* Retrofit for API
* (Planned) Room for local database

---

## 🛠️ Tech Stack

* Java (Android)
* Retrofit (API calls)
* LiveData / ViewModel
* Material Design Components
* AlarmManager (Adhan scheduling)
